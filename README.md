# Real-Time Chat Application

A real-time chat application built to explore the fundamentals of networking, server design, and real-time communication. The project focuses on implementing the core mechanics behind messaging platforms while gaining hands-on experience with backend development and system architecture.

---

## Overview

This application allows multiple users to connect to a central server and exchange messages instantly. The server handles client connections, message routing, and chat room management while using structured JSON objects for communication.

The primary goal of this project is to understand how modern chat systems work internally rather than relying heavily on large frameworks.

---

## Planned Features

* Real-time messaging between connected users
* Chat room support
* Server-side message routing
* Structured message format using JSON
* User account handling
* Message storage and retrieval

---

## System Architecture

The application follows a client-server architecture.

**Server Responsibilities**

* Accept and manage client connections
* Route messages to the correct recipients
* Manage chat rooms
* Handle user accounts
* Store and retrieve messages

**Client Responsibilities**

* Connect to the chat server
* Send messages in JSON format
* Receive and display incoming messages

---

## Technologies

The technologies used in this project include:

* TypeScript / JavaScript
* Node.js
* WebSockets for real-time communication
* JSON for message formatting
